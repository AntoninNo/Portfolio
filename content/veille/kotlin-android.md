Title: Forte adoption de Kotlin sur Android
Date: 2024-01-01
Tags: Kotlin, Android, popularité
Summary: Kotlin dépasse Java sur Android
Source: https://www.androidmetro.com/2024/01/android-kotlin-vs-java-market-share.html

Kotlin représente aujourd’hui environ 70 à 75 % des usages sur Android, dépassant Java. Il est devenu le langage majoritairement utilisé pour les nouveaux projets mobiles.
