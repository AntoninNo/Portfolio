Title: Kotlin vs Java sur Android
Date: 2024-01-01
Tags: Kotlin, Java, comparaison
Summary: Kotlin plus productif que Java
Source: https://www.androidmetro.com/2024/01/android-kotlin-vs-java-market-share.html

Kotlin permet d’écrire jusqu’à 40 % de code en moins que Java et réduit les erreurs courantes. Cela explique pourquoi il est aujourd’hui préféré pour les nouveaux projets Android.
